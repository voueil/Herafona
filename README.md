# Herafona
Herafona is a web platform that allows users to explore and book traditional Saudi craft workshops, connecting them with local artisans while preserving Saudi heritage

## Folder Structure
Herafona/
│
├── index.html        
├── style.css         
├── app.js            
└── package.json     

## Pages Overview
| Section | Description |
|----------|-------------|
| **Home Page** | Displays all available workshops with their details and booking options. |
| **Search Bar** | Allows users to search locally for workshops by name, city, artisan, or description. |
| **Booking Form** | Lets users enter their name and email to confirm a booking. |
| **Chatbot** | Integrates IBM Watson Assistant for user interaction and quick help. |


## Technologies Used
- **HTML** & **CSS** – Structure and design
- **IBM Watson Assistant** – Chatbot integration for user support   
- **JavaScript** – Dynamic rendering and interactivity  
- **Visual Studio Code** – Development environment  


## How It Works
1. The user browses available workshops on the homepage.  
2. The search bar filters workshops by relevant keywords.  
3. Clicking **Book Now** opens a booking form.  
4. The user enters their name and email to confirm.  
5. A confirmation alert appears with booking details.  
6. users can interact with the chatbot for assistance.
